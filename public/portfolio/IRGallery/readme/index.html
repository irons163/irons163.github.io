Features

  • Single tap fullscreen mode.
  • Double tap image zooming.
  • Pinch zooming.
  • Captions.
  • Rotation support.
  • Load images locally or from a web URL.
  • Custom UITabBarItems.
  • Default UITabBarItems:
    • Delete.
    • Favorite.
    • Share.
