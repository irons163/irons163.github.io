  1. Socket environment initial.
  2. Parse the input parameters.
  3. CreateClient. Get class RTSPClient construct (return class medium and some vars)
  4. Send RTSP “options“ and get OPTIONS from server.
  5. Get SDP description by the URL of the server(return value:SDPstring)
  6. Create media session from the SDP descriptor above. For each subsession, RTSPClient->setupMediaSubsession(*) 7 Create output files: Only for the Receiver (Store the streaming but not play it) For different file format, use different *FileSink class This uses the QuickTime file as demo. Output to the ‘::stdout’
  7. startPlayingStreams
